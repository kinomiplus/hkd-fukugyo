# X戦略・マネタイズ方針

## 全体戦略
X投稿で認知を広げ、noteで信頼を築き、
Studio LPでアフィリエイト収益を得る。

## 導線
X投稿 → noteフォロワー獲得 → Studio LP → アフィリエイト登録 → 収益

## X投稿戦略
- 投稿数：1日5本
- 投稿時間：7時・12時・17時・20時・22時
- 投稿テーマ優先順位：
  1. 北海道特化の副業・転職情報
  2. 実体験・体験談
  3. お役立ち情報
  4. 共感系・悩み系
  5. Studio LPへの誘導

## note戦略
- 投稿数：週2〜3本
- 無料記事：有益情報でフォロワー獲得
- 有料記事：500〜1,000円で販売
- 記事末尾に必ずStudio LPへのリンクを貼る
