# チーム2：教育AI（note担当）

## チーム目的
信頼構築・SEO流入・有料記事販売。
noteを軸に読者を教育し、アフィリエイト成約と有料記事販売につなげる。

## 担当社員
| 社員 | 役割 |
|------|------|
| 社員5（05_post） | note無料記事の生成（週2〜3本） |
| 社員6（06_product） | note有料記事の構成・本文・セールスレター生成 |

## チーム強化方針
- 記事の最後に必ずStudio LP誘導を入れる（1〜2行、宣伝臭ゼロ）
- シリーズ記事（3〜5本完結）で継続読者を増やす
- 無料記事の中に有料記事への導線を自然に入れる
- 北海道特化キーワードでSEO上位を狙う

## シリーズ記事の設計ルール
- テーマ例：「北海道副業スタートガイド全5回」「転職成功ロードマップ全4回」
- 各回の末尾に「次回：〇〇」と予告を入れ、継続読者を確保
- シリーズ1本目は無料、2〜3本目は有料にする導線設計を基本とする
- タイトルに「第〇回」「〜編」を入れてシリーズ感を出す

## SEOキーワード戦略
優先キーワード（全記事のいずれかに必ず含める）：
- 北海道 副業
- 北海道 転職
- 札幌 副業 在宅
- 北海道 在宅ワーク
- 北海道 フリーランス

ロングテール狙い：
- 「北海道在住 副業 おすすめ」
- 「札幌 転職 エージェント 評判」
- 「北海道 副業 会社員 月5万」

## 有料記事への導線ルール（無料記事内）
- 本文中盤に「この続きは有料記事で詳しく解説しています」と1回入れる
- 記事末尾に有料記事へのリンクと一言説明を入れる
- 導線文は売り込み感なし。「より深く知りたい人向け」のトーンで

## LP誘導文のテンプレート（記事末尾に必ず入れる）
```
北海道在住で副業・転職を考えている方は、
ぼくがまとめたスタートガイドも参考にしてみてください。
→ [プロフィールのリンクから確認できます]
```

## Slack報告ルール
- 報告チャンネル：#team2-note-report
- 頻度：記事公開のたびに（週2〜3回）
- 報告フォーマット：
  ```
  【教育チーム投稿報告】yyyy/mm/dd
  ■ 記事タイトル：〇〇
  ■ 種別：無料 / 有料
  ■ シリーズ：あり（〇〇シリーズ第〇回）/ なし
  ■ SEOキーワード：〇〇、〇〇
  ■ 有料記事への導線：あり / なし
  ■ LP誘導：あり / なし
  ■ 次回予定記事：〇〇
  ```
