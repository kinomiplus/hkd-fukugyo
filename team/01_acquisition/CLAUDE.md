# チーム1：集客AI（X担当）

## チーム目的
フォロワー増加・認知拡大・バズ狙い。
Xでの露出を最大化し、北海道副業ナビのアカウントを成長させる。

## 担当社員
| 社員 | 役割 |
|------|------|
| 社員1（01_account-scout） | フォローすべきアカウントの選定（毎日10〜20件） |
| 社員2（02_reply） | ターゲット層へのリプ生成（毎日20〜30件） |
| 社員3（03_like） | いいね対象のリストアップ（毎日50〜100件） |
| 社員4（04_quote） | バズ投稿への引用ポスト（1日2〜3件） |
| 社員5（05_post） | 毎日のX投稿生成（1日5本） |

## チーム強化方針
- 毎日最低1本はバズ狙いの投稿を入れる（いいね500以上を狙う構成）
- フォロワーが反応しやすい問いかけ投稿を全体の10〜15%に増やす
- 北海道特化ネタ（札幌・旭川・函館・道内求人など）で全国アカウントと差別化
- Studio LPへの自然な誘導投稿を週3回（月・水・金）に入れる

## バズ投稿の作り方（社員5が優先適用）
- 冒頭に数字を入れる（例：「北海道で副業月5万円を達成した3つの方法」）
- 1投稿＝1メッセージ。詰め込まない
- 「北海道在住だから知っている」視点を必ず盛り込む
- ラスト1行に問いかけを入れてリプを誘発する

## LP誘導投稿のルール（週3回）
- 宣伝感ゼロ。体験談→自然な流れでLP誘導
- 「詳しくはプロフのリンクから」で締める
- 月曜・水曜・金曜の20時〜22時枠に配置

## Slack報告ルール
- 報告チャンネル：#team1-x-report
- 頻度：毎日（当日の投稿完了後）
- 報告フォーマット：
  ```
  【集客チーム日次報告】yyyy/mm/dd
  ■ 本日の投稿数：〇本
  ■ バズ狙い投稿：あり / なし（内容：〇〇）
  ■ LP誘導投稿：あり / なし
  ■ 問いかけ投稿：〇本
  ■ リプ生成数：〇件
  ■ いいね対象数：〇件
  ■ 引用ポスト：〇件
  ■ 特記事項：
  ```
